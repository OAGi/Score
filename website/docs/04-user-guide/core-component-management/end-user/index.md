---
title: "End User Core Component Management"
sidebar_position: 0
---

connectCenter allows end users to create their own Core Components and use them during the BIE extension.
That is, users are not limited to using existing, standard CCs but also user-defined Core Components.

To manage end user core components, make sure that a published release branch (not the Working branch or a draft release branch) is selected on the "Core Component" page.
