---
title: "Non-standard Namespace Management"
sidebar_position: 1
---

All EUCCs require a Non-standard namespace (i.e., end user namespace).
Standard Namespaces (i.e., developer namespace) cannot be used in an EUCC.

An end user can [create](../developer/02-namespace-management.md#create-a-namespace) and [edit](../developer/02-namespace-management.md#edit-detail-of-a-namespace) a Namespace he/she owns.
He/she can also [find](../developer/02-namespace-management.md#find-a-namespace) and [view](../developer/02-namespace-management.md#view-detail-of-a-namespace) the details of a Namespace.
Finally, he can [discard](../developer/02-namespace-management.md#discard-a-namespace) a Namespace or [transfer the ownership](../developer/02-namespace-management.md#transfer-ownership-of-a-namespace) of a Namespace to another user.
It should be noted that the transfer of the ownership of a Non-standard namespace can occur only between end users.
