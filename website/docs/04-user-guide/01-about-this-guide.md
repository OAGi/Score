## How to use this guide

Use the menu on the left to browse through sections.

The source of this guide is split into multiple pages, listed in the menu.
Use the search box at the top of the page to find content in this guide.

Due to limited development resources, the user guide is designed to minimize its maintenance due to UI changes.
Therefore, it contains minimal UI graphics.
It is recommended that the user guide is read along with the tool.
Follow the instruction along with the tool interactions.

## Nomenclature

| Structural Format | Meaning |
|---|---|
| "Text in Double Quotes" | Double quoted text is used for button labels, labels, table column names on a page, names of web pages, label of a tree node, and example text for a form field. |
| Italicized Text | Italicized text is used to form field labels or emphasize some keywords. |
