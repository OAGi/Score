Filename Syntax: 
expression       = { literal | optional-group | placeholder } ;
optional-group   = "(" , expression , ")" ;
placeholder      = "{" , placeholder-name , [ "?" , flag-name ] , [ ":" , modifier-chain ] , "}" ;
modifier-chain   = modifier-token , { ":" , modifier-token } ;
modifier-token   = "separator" , "(" , quoted-arg , ")"
                                 | "replace" , "(" , quoted-arg , "," , quoted-arg , ")"
                                 | raw-separator ;
Duplicate Handler Syntax: -{Incremental}, ~{BIE ID}
Flags: `includeBusinessContext`, `includeVersion`
Common placeholders: `BIE Property Term`, `BIE Display Name`, `BIE Version`, `BIE DEN`, `BIE Status`, `BIE Remark`, `BIE ID`, `Business Context Names`, `Business Context Name[0]`
BIE Package only: `BIE Package Name`, `BIE Package Version ID`, `BIE Package Version Name`
Duplicate handler placeholders: `Incremental`, `BIE ID`
Example (BIE Schema): {BIE Property Term:separator('-')}(-{Business Context Name[0]?includeBusinessContext})(-{BIE Version?includeVersion:replace('\\.', '_')})
Example (BIE Package Schema): {BIE Package Name}-{BIE Package Version ID}_{Business Context Names:replace('\\s+', ''):separator('+')}_{BIE Property Term}([{BIE Display Name}])(-{BIE Version})
