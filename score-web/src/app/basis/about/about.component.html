Score

Score is a software tool that supports the development of data exchange standards based on the ISO 15000-5 Core Components standard. Score provides a repository and functionalities for standards developers to create, publish, and maintain data exchange standards with higher precision, quality, and productivity. Standards are created once in the syntax/platform-independent form and can be published in various implementation-specific languages such as XML Schema, JSON Schema, or Open API specification. In addition, Score allows the standard users to create (in a syntax/platform-independent form) a standards usage model (also known as implementation guide) for specific integration scenarios, and then serialize each into desired implementation-specific languages.

You can find out more information at the following link: Score: Standards Life Cycle Management Tool
