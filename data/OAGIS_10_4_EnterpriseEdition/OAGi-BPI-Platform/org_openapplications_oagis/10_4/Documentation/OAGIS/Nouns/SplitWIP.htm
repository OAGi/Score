Element: SplitWIPDestination

Name SplitWIPDestination
Type SplitWIPDestinationType
Nillable no
Abstract no
Documentation Information concerning the specific production order in the WIP Split transaction to be processed. Instances of this component are the target (children) production orders to which to perform the split transaction. If the parent production order (specified in the SplitWIPSource component) is also a resulting child production lot, it must be specified again as a child production order (ie. you must define the SplitWIPSource again as an instance in the SplitWIPDestination).
More information at: http://www.openapplications.org/oagis/10.
