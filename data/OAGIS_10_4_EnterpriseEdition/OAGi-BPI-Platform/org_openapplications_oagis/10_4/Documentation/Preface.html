Preface

This document continues the work of the Open Applications Group Specification Teams since they were first formed in the fall of 1995.  It reflects many hundreds of person hours contributed by the Open Applications Group, Inc. member companies. 

 These people represent the persons who are building, testing, and implementing the business applications thousands of companies are using world wide to run their businesses.  Many customers and other members of the global software community have also contributed to this effort.

OAGIS release is not the sum total of all of the interoperability points between business applications but it adds some important and exciting new enhancements including:

  • The basis for Mobile, Cloud, JSON Work.

  • The Model

  • Chem eStandards Equivalence

  • Master Scenarios

  • HR-XML Harmonization

  • Manufacturing

  • Enhanced and Improved Extension Mechanisms

  • Single Namespace for faster and easier implementation and easier to use

  • UN/CEFACT CCTS and NDR 3.0 compliance

  • Local Elements for improved Contextualization Capability

There is a great deal more work to be done and this document will continue to be updated.  Much care has been given to make it as correct as possible.  If you find any errors, or if you have any comments, please do not hesitate to contact us at oagis@openapplications.org.. 

Since each chapter of this document is viewable through a browser and provides hyperlink throughout the documentation it is necessary to unzip the entire zip file in a directory and maintain the hierarchy provided in the zip file. Breaking this directory structure will give you unpredictable results. Additional information about OAGIS can be obtained from our Web Site at www.openapplications.org, as well as, more information about the current release of OAGIS, OAGIS release.

 We here at the Open Applications group hope you will find this document useful.
